For the browser

Download the browser-based version of Maker.js, then upload it to your website:

http://microsoft.github.io/maker.js/target/js/browser.maker.js

Or you can link to it:

<script type="text/javascript" src="http://microsoft.github.io/maker.js/target/js/browser.maker.js"></script>

Then, in your JavaScript, use the require function to get a reference:

var makerjs = require('makerjs');

More tutorials are coming soon! Meanwhile you can check the GitHub repo examples.
