--- title: Models ---

A model is represented by an object with this mandatory property:

  • id: string

A model may have these optional properties:

  • origin: point
  • paths: array of paths
  • units: string
  • models: array of models

Let's look at paths first, using the example above.

{% highlight javascript %} var makerjs = require('makerjs'); var line = { id: 'myLineId', type: 'line', origin: [0, 0], end: [50, 50] }; var circle = { id: 'myCircleId', type: 'circle', origin: [0, 0], radius: 50 }; var pathArray = [ line, circle ]; var model = { paths: pathArray }; var svg = makerjs.exporter.toSVG(model); document.write(svg); {% endhighlight %}
