--- title: Models ---

A model is represented by an object with this mandatory property:

  • id: string

A model may have these optional properties:

  • origin: point
  • paths: array of paths
  • units: string
  • models: array of models

Let's look at paths first, using the example above.

{% highlight javascript %} var makerjs = require('makerjs'); var line = { id: 'myLineId', type: 'line', origin: [0, 0], end: [50, 50] }; var circle = { id: 'myCircleId', type: 'circle', origin: [0, 0], radius: 50 }; var pathArray = [ line, circle ]; var model = { id: 'myModel', paths: pathArray }; var svg = makerjs.exporter.toSVG(model); document.write(svg); {% endhighlight %}

Note the we can also pass a model to makerjs.exporter.toSVG.

If we wrap our model code in a function we can call it multiple times. There are several ways to do this. First we will leave the code as is, and return the model variable.

{% highlight javascript %} var makerjs = require('makerjs'); function myModel() { var line = { id: 'myLineId', type: 'line', origin: [0, 0], end: [50, 50] }; var circle = { id: 'myCircleId', type: 'circle', origin: [0, 0], radius: 50 }; var pathArray = [ line, circle ]; var model = { id: 'myModel', paths: pathArray }; return model; } var svg = makerjs.exporter.toSVG(myModel()); document.write(svg); {% endhighlight %}

Alternatively, we can change our function to be usable with the new operator, and our model properties are set using the this keyword:

{% highlight javascript %} var makerjs = require('makerjs'); function myModel() { var line = { id: 'myLineId', type: 'line', origin: [0, 0], end: [50, 50] }; var circle = { id: 'myCircleId', type: 'circle', origin: [0, 0], radius: 50 }; var pathArray = [ line, circle ]; //set properties using the "this" keyword this.id = 'myModel'; this.paths = pathArray; } //note we are using the "new" operator var svg = makerjs.exporter.toSVG(new myModel()); document.write(svg); {% endhighlight %}

The example output should be the same as above. While we changed the way we defined our model, we haven't yet changed the functionality:

Now we are better set up to look at models and origin. We will create a new model which has 2 instances of our model:

{% highlight javascript %} var makerjs = require('makerjs'); function myModel() { var line = { id: 'myLineId', type: 'line', origin: [0, 0], end: [50, 50] }; var circle = { id: 'myCircleId', type: 'circle', origin: [0, 0], radius: 50 }; var pathArray = [ line, circle ]; //set properties using the "this" keyword this.id = 'myModel'; this.paths = pathArray; } var model1 = new myModel(); var model2 = new myModel(); //they will be on top of each other, so let's move the origin model2.origin = [100, 0]; var model = { id: 'iHave2models', models: [ model1, model2 ] }; var svg = makerjs.exporter.toSVG(model); document.write(svg); {% endhighlight %}
