--- title: SVG Styling ---

It may be helpful to add visual styles to your drawing when debugging. You can do this using CSS and reference elements in your SVG. Of course this will only add style in the browser, it will not affect SVG exported to a file for the laser cutter.

The keys you use within your paths and models collections become an id attribute in the generated SVG. For Example:

{% highlight javascript %} var makerjs = require('makerjs'); var model = { paths: { circle1: new makerjs.paths.Circle([0, 0], 30) }, models: { ovalArc1: new makerjs.models.OvalArc(120, 60, 50, 10) } }; var svg = makerjs.exporter.toSVG(model); document.write(svg); document.write('
