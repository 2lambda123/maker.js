--- title: Rotating ---

You can rotate paths and models with these functions:

  • makerjs.path.rotate(path: object, angleInDegrees: number, rotationOrigin: point)
  • makerjs.model.rotate(model: object, angleInDegrees: number, rotationOrigin: point)

Each of these functions return the original object, so that we can "chain" on the same line of code.

Rotate path example:

{% highlight javascript %} var makerjs = require('makerjs'); var line1 = new makerjs.paths.Line('line1', [0, 0], [100, 0]); var line2 = new makerjs.paths.Line('line2', [0, 0], [100, 0]); var paths = [line1, makerjs.path.rotate(line2, -30, [100, 0])]; var svg = makerjs.exporter.toSVG(paths); document.write(svg); {% endhighlight %} try it in JsFiddle ⇗

Rotate model example:

{% highlight javascript %} var makerjs = require('makerjs'); var rect1 = new makerjs.models.Rectangle('rect1', 20, 40); makerjs.model.rotate(rect1, 45, [0, 0]); var svg = makerjs.exporter.toSVG(rect1); document.write(svg); {% endhighlight %} try it in JsFiddle ⇗
