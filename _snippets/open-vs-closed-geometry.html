--- title: Open vs Closed Geometry ---

Maker.js works with both open and closed geometries. However as we will soon see, with closed geometries there is additional functionality.

We briefly discussed chains and endless chains, now we will see how they relate to open and closed geometries.

Examples of Open Geometry:
  • A solitary line path.
  • A solitary arc path.
  • A chain of paths (with dead ends).
{% highlight javascript %} //an asterisk shape that is not a closed geometry var makerjs = require('makerjs'); function asterisk() { function plus() { this.paths = { h: new makerjs.paths.Line([-5, 0], [5, 0]), v: new makerjs.paths.Line([0, -5], [0, 5]) } } this.models = { plus: new plus(), plus1: makerjs.model.rotate(new plus(), 45) }; } var svg = makerjs.exporter.toSVG(new asterisk()); document.write(svg); {% endhighlight %} Examples of Closed Geometry:
  • A circle path.
  • An endless chain of paths.
  • Any of the above, nested within another and not overlapping.

The house, tablet mount, circular adapter plate, and skateboard deck are good examples of a closed geometry. They have nested endless chains (such as circles, squares, or rounded rectangles). If you were to print them on a laser cutter, you would have a complete shape.
